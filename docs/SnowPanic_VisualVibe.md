# SnowPanic: Visual Vibe & Avalanche Reference

このドキュメントは、SnowPanic の開発において「目指すべき最高に気持ちいい雪崩のビジュアルと感覚」を忘れないための公式リファレンスメモです。

## 📸 参考ビジュアル（Reference）
**元ネタ:** 共有された動画スクリーンショット「雪国名物!! 気持ちいい瞬間ベスト5」
* 下からスコップで屋根の雪を「トン」と突いた直後に、屋根全体に乗っていた巨大な雪が一体となってドドドドッと崩れ落ちる極めて爽快な映像。

## 🎯 達成すべきコア要素（The Core "Vibes"）

### 1. 「トン」という正確無比なトリガー（The Perfect Trigger）
- **感覚:** 下からの軽い衝撃一つで、限界まで持ちこたえていた均衡が破れる快感。
- **実装要件:** 手袋のタップが、屋根の座標と完全に一致している必要がある。ズレて「空振り」や「無反応」が起きるとこの快感は一瞬で殺されてしまう。

### 2. 面としての巨大なスライド（The Cohesive Slab Slide）
- **感覚:** バラバラの粉雪がパラパラ落ちるのではなく、分厚い「一枚の超巨大な雪玉・板」が自重で一気に迫り出してくる重厚感。
- **実装要件:** 落ち始めは細かな破片ではなく、屋根の面積を保ったままズレ動く表現（あるいは大きめの塊の連鎖）が求められる。

### 3. 空中での豪快な崩壊と着弾の重量感（The Massive Crumble & Impact）
- **感覚:** 軒先（Eave）から空中に押し出された巨大な塊が、自重でグニャッと曲がり、耐えきれずにド派手に崩落していくカタルシス。
- **実装要件:** 
  - `SpawnAvalancheBurstVisual` で落とす雪塊（現在 `PrimitiveType.Sphere`）をもっと平たく、かつ巨大な塊（Chunks）にする。
  - 地面に激突した瞬間に、重量感を感じさせるための「パウダースノーの飛沫（雪煙パーティクル）」を盛大に発生させる。
