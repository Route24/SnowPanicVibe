# SNOW PANIC - CORE DESIGN

【ゲームの本質】
Angry Birds のような「シンプル物理アクション」

【プレイヤーの行動】
- タップ（クリック）で雪を壊す

【ゲームの目的】
- 屋根の雪をすべて落とす / 壊す

【最重要コア】
1. 屋根の上にオブジェクトがある
2. プレイヤーがクリックする
3. その対象が消える or 物理的に崩れる
4. 連鎖・崩れが気持ちいい

【絶対ルール】
- 見た目とロジックは分離
- クリック判定は共通システム
- 雪の種類（A/B）はデータ差し替えで対応
- シーン依存の直書き禁止

【今のフェーズ】
- **Phase 0**
  - 1個のオブジェクトをクリックで消せることだけを完成させる
  - （＝現在進行中の ANTI PROTOCOL）
- **Phase 1**
  - 複数配置
  - 連続クリック
- **Phase 2**
  - 物理崩れ
