# 雪表現の設計方針

## 基本原則

**「見た目は粒っぽく、内部ロジックは軽量」**

## 適用ルール

| 領域 | 方針 |
|------|------|
| 屋根上の雪管理 | 軽い塊/セルベースを優先 |
| シミュレーション | フル粒子シミュレーションは不要 |
| 演出 | 崩壊時と落下時だけ粒っぽい演出を足す |

## 実装時の指針

1. **屋根上**: 塊やセルのデータで管理。パーティクルは表示用に最小限。
2. **崩壊時**: 叩いた瞬間・塊が分かれる瞬間に粒感を出す。
3. **落下時**: 軒先落下・地面到達時など、動きが目立つタイミングで粒っぽい演出。
4. **常時**: 全フレームで物理粒子を追跡しない。
